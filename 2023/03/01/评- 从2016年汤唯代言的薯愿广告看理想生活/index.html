评- 从2016年汤唯代言的薯愿广告看理想生活
Mar 1, 2023

原来

2015年，数手指算算是初三。娱乐活动被管控得极严，很少能获得观看电视的机会。除非一个写着作业的晚上，从房间借着喝水放松的接口踱出——往往这时电视剧要恰好放到两集中间的广告，所以原本沙发上的家人都去厨房或者卧室各忙各务——我就在这样无人监管的空隙趁机在客厅徘徊——大脑放空的时候，电视里的下集预告是最好的消遣。

当然大部分时间只能看得到广告。

在漫长求学生涯中极度缺少记忆点的一年，我乏善可陈的生活中突然闯入这样一段有声有色的鲜活片段，大概怎么都是要在脑海中深烙一笔的。

本来

广告原片

后来

首先的后果是这个广告成功地在我身上发挥了它应有的效用。从那以后，凡去超市，必买薯愿，必须是蜂蜜牛奶味。

我固执地（甚至现在也）认为这款零食吃了就能和汤唯一样，无忧无虑的快乐多到溢出取景框。事实上好像吃的时候也真的短暂地成功了。虽然一口气吃掉一盒稍稍有点腻，需要喝一杯水来缓解，但是那些咀嚼的瞬间我仿佛忘掉了吃完还要去写的那些写也写不完的作业，青春期赶也赶不走的烦恼。我模仿女明星在荧幕上轻快的咬合动作，发出清脆的噪音：

“咔吱咔吱咔吱咔吱咔吱……”

这使人着迷，深陷其中，无法自拔——反正我的自控力不会允许我终止这种类似，服药的行为。那么当然不知不觉中饼干就吃完了——不会有其他的结果。

“吃一点尝尝还不够？一张嘴刹不住车？”
=> “眼里没数！缺乏自控力！抵挡不住诱惑！”

“只想着自己贪吃？不给其他人分享？谁花钱给你买的？”
=> “眼里没有别人！自私自利！忘恩负义！”

“餐前吃零食？吃饭食欲不振？”
=> “挑食任性！” => “眼里没有父母！不尊重劳动！” => “不知感恩！”

“垃圾还不赶紧拿走？吃得一桌子碎渣不拿抹布擦？”
=> “邋遢！没素质！破坏干净！” => “眼里没有这个家！从不主动打扫卫生！懒惰！”

对了，所有的判定最后只能推导出一个结论：不孝逆子。

所以后续的不满和说教的也顺利成章地反复发生。然而事实上并不是“后续的”。

一旦撕开塑料包装袋的声音惊动了世界，这个世界就没有消停过。

管他呢，这不妨碍蜂蜜牛奶味薯愿还是我最爱吃的零食。而且随着年龄的增长，从启封的一刹那，我愈发能全情投入在唇齿舌间，谁还顾得上听那聒噪模糊的背景音是平是仄？内心只余平和与安宁，再咂摸还有一丝甜蜜。

这款零食从此多了一个木头追随者。每见必买，不管是不是真的想吃，也许现在早就不觉得它有多美味，但无穷的快感是一定的，像针锥刺发麻的大腿。

从来

在新拍摄的广告中，汤唯伴随着轻快的音乐，在屋子中跳起了一代传奇巨星张国荣的经典“阿飞舞”，身着家居服的她在看似的随意中却从内到外透露着女神的自然与自信。而她手中捧着的“主角”薯愿，在唇齿间发出清脆的响声，非油炸焙烤的薯片带来独特的味道和口感，难怪女神会对它爱不释手，而那句“我只要薯愿，尝尝”更带给人一种难以拒绝的诱惑。

周末大扫除完毕，轻扬的窗帘，午后的日光，穿着棉布家居服倒在欧式铁架床上听无线电。阳台上刚刚晾晒的衣服一件一件荡开，空气中若有若无地飘散着薰衣草洗衣粉的味道。这个时候猫也不知道躲去哪个角落打盹，消磨宅家的时间。闹市的老公寓，温馨的房间不大刚好一人居，极繁主义、爱好广泛、生活充实，但是多而不乱。这份享受闲暇时光的从容折射出画面主人公独立、自信的特质，有稳定的职业还有能力饲养宠物和爱好。

将来

这个场景可能是最理想的生活状态。

一个虚无缥缈的立方体，在梦河里打着旋儿慢慢悠悠地漂着。不记得是一个还是多个影视印象的糅合了，这些年越来越来清晰，既怀疑又笃定。
